# Agencia de viajes Easy Peasy Travel

Proyecto Php de Alejandro Vivas López

## Instalación

Ejecutar en HeidiSql o tu editor sql los ficheros:

- database.sql
- datos.sql

Cambiar en vistas/conexion_db.php las credenciales de conexión.
